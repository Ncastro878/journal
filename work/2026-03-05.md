# Slack Ticket Bot - Staging Deployment & Debug

**Project:** eCustom-Solutions/jobs-service
**PRs:**
- Dev: https://github.com/eCustom-Solutions/jobs-service/pull/714
- Staging: https://github.com/eCustom-Solutions/jobs-service/pull/716

Built and deployed a Slack ticket bot system for Quill Lemon attorneys to create Disbursement/SRED ticket requests without needing HubSpot access. The system creates HubSpot tickets, resolves ticket owners via polling, and manages an approval workflow between ticket owners and attorneys.

## Key Changes
- Slack bot with Socket Mode for real-time interactions
- Modal-based ticket creation with pipeline selection
- Automatic ticket owner resolution by polling HubSpot
- Notification workflow with approve/reject/send back actions
- Notes system that accumulates history for attorneys
- Deal association by Clio Matter ID lookup
- HubSpot notes added to ticket timeline for audit trail

## Deployment Debug
- Created separate staging branch via cherry-pick for independent deployments
